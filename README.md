# 🎬 Indica AI

O **Indica AI** é uma aplicação desenvolvida em **React** que recomenda filmes com base no gosto do usuário.

Primeiro, o usuário escolhe um gênero de interesse. Em seguida, são exibidos filmes desse gênero para que ele informe se gostou ou não de cada um. Após avaliar três filmes, a aplicação utiliza Inteligência Artificial para analisar as preferências e gerar recomendações personalizadas de filmes semelhantes.

---

## ✨ Funcionalidades

- 🎭 Escolha entre diferentes gêneros:
  - Terror
  - Ação
  - Animação
  - Comédia
  - Romance

- 🎥 Exibição dinâmica de filmes do gênero selecionado.
- 👍👎 Avaliação dos filmes através de "Gostei", "Não gostei" ou "Não Vi".
- 🤖 Recomendações personalizadas utilizando Inteligência Artificial.
- ⏳ Tela de carregamento durante a geração das recomendações.
- 📱 Interface moderna e responsiva.

## 🔗 APIs utilizadas

### TMDB API

Responsável por buscar os filmes(nome, capa e sinopse) de acordo com o gênero escolhido pelo usuário.

### Gemini API

Após o usuário avaliar três filmes, a aplicação envia essas informações para a IA, que analisa as preferências e retorna recomendações de filmes semelhantes.

## 🚀 Como executar

1. Clone o repositório

```bash
git clone https://github.com/SEU-USUARIO/Indica-AI.git
```

2. Instale as dependências

```bash
npm install
```

3. Crie um arquivo `.env`

```env
VITE_GEMINI_API_KEY=SUA_CHAVE
VITE_TMDB_API_KEY=SUA_CHAVE
```

4. Execute o projeto

```bash
npm run dev
```
